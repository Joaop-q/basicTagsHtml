Lista Ordenada

A tag <ol> cria uma lista numerada. Cada item é definido com <li>.

  1. Primeiro item
  2. Segundo item
  3. Terceiro item

Lista Não Ordenada

A tag <ul> cria uma lista com marcadores. Cada item é definido com <li>.

  • Item A
  • Item B
  • Item C

Abreviação

A tag <abbr> mostra o significado de uma abreviação ao passar o mouse.

HTML é a linguagem de marcação da web.

Citação

A tag <blockquote> é usada para destacar uma citação.

Link

A tag <a> cria um hiperlink para outra página ou site.
